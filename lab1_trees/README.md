# Работа с деревом.
## Возможности
- Поддержка операций:
- создание дерева(корня);
- добавление узла;
- вывести созданное дерево;
- удалить дерево;
- проверить монотонно ли возрастает ширина.

## Основные функции
- createtree (создание дерева)
- printTree (вывод дерева)
- insertNode (добавление узла)
- deleteTree (удалить дерево)
- treemono (проверить монотонность увеличения ширины)
